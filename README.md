# YALLA BAU Bus Tracking App

YALLA BAU is a Flutter-based university bus tracking system.

## Features
- Student and Driver Login
- Live Bus Tracking
- Route and Stops Display
- Bus Capacity Status
- Real-time Trip Updates
- Firebase Authentication
- Firestore Database

## Technologies Used
- Flutter
- Dart
- Firebase
- Cloud Firestore
